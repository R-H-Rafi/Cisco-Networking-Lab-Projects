# Cisco Networking Lab Projects

A collection of Cisco Packet Tracer networking projects demonstrating practical implementations of LAN, WAN, routing, and network services. These projects were developed as part of networking practice and academic learning to strengthen skills in network design, routing protocols, and enterprise infrastructure deployment.

## Technologies & Concepts Covered

- Cisco Packet Tracer
- LAN Design
- WAN Connectivity
- Static Routing
- Dynamic Routing (RIP, OSPF)
- DNS Configuration
- DHCP Configuration
- Email Server Configuration
- FTP Server Configuration
- Web Server Configuration
- Router and Switch Configuration
- Enterprise Network Architecture
- Multi-Site Network Design
- Inter-LAN Communication

---

## Projects Included

- DNS Server
- Mail Servers
- Email Communication
- Domain Name Resolution
